This project was developed as part of the SkillCraft Technology Cyber Security Internship (Task 02).

The objective was to create an image encryption tool that protects image data through encryption and decryption techniques. 
While the task suggested basic pixel manipulation, this implementation uses the Advanced Encryption Standard (AES) algorithm to
provide stronger security for image files.

The project allows users to:

Upload an image
Encrypt the image using AES
Save the encrypted output
Decrypt the image using the correct key
Recover the original image

Why AES?

The internship task suggested implementing encryption through pixel manipulation or simple mathematical operations.
To explore a more robust approach, this project implements the Advanced Encryption Standard (AES), which is a widely used symmetric
encryption algorithm for protecting digital data.
